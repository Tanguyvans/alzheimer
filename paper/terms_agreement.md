## NACC/SCAN (National Alzheimer's Coordinating Center / SCAN Neuroimaging)

### Relationship
- **NACC** = Clinical/tabular data from 40+ ADRCs
- **SCAN** = NACC's neuroimaging repository (MRI, PET scans)
- If you used MRI from NACC portal, you used SCAN data

### Requirements
- [x] **Methods description**: Clarify that MRI data from NACC comes from SCAN
- [x] **Acknowledgment NACC**: Include grant U24 AG072122
- [x] **Acknowledgment SCAN**: Include U24 grant AG067418 + ALL contributing ADRCs
- [ ] **Pre-submission review**: Submit to NACC Publications System before journal submission
- [ ] **Review timeline**: Up to 5 business days
- [ ] **NIH compliance**: If NIH funded, submit to PubMed Central (PMC)
